# FinGuard Fraud Detection

Enterprise-level Credit Card Fraud Detection System using Machine Learning, XGBoost, SHAP, and Streamlit.

## Features
- Data preprocessing and scaling
- Fraud pattern visualization
- Multiple ML model comparison
- XGBoost optimization
- SHAP explainability
- Real-time fraud prediction dashboard

## Tech Stack
Python, XGBoost, SHAP, Scikit-learn, Streamlit

## Dataset
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Run Project
pip install -r requirements.txt
python src/models.py
streamlit run dashboard/app.py
